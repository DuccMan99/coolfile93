[5.1] When is a batsman out LBW?

           -----
          |  |  |
          |  |  |
off <-----|  |  |----> leg stump(for a right-hander)
stump   --|--|--|--
       |__|__|__|__|
       |           |
  -----|-----------|-------
       |  |     |  |
       |  |     |  |
       |A |  B  |C |
       |  |     |  |
       |  |     |  |

A = Outside off stump
B = In-line with the stumps
C = Outside legstump
